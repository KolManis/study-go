# Конспект: Словарь (Ассоциативный массив)

## Основное понятие
- **Словарь** (map, dictionary) — структура данных для хранения пар «ключ-значение»
- **Ключ** — уникальный идентификатор
- **Значение** — может быть любой переменной (даже другой структурой данных)
- **Неупорядочен** — порядок вывода элементов не гарантирован

## Варианты словарей
- **Обычный словарь** — каждый ключ уникален
- **Мультисловарь** (multimap) — позволяет хранить несколько значений для одного ключа

## Основные операции (при реализации на хеш-таблицах)
- **Добавление** элемента — O(1)
- **Удаление** по ключу — O(1)
- **Изменение** значения — O(1)
- **Получение** значения — O(1)

## Применение
- Хранение конфигураций
- Частотные словари в тексте
- Быстрые поисковые индексы
- Подсчёт и группировка данных
- Хранение вложенных структур

## Реализация
Чаще всего реализуется через **хеш-таблицы**, что обеспечивает константное время операций.
