# Подсолнечник

**Ограничение времени:** 1 с  
**Ограничение памяти:** 64.0 Мб  


---

## Условие задачи

Дано семейство множеств $S_1, S_2, \dots, S_n$.  
Требуется определить, образуют ли они **подсолнечник**: то есть существует такое множество $C$ (ядро), что:

1. Для любых двух различных $i \neq j$ выполняется $S_i \cap S_j = C$
2. «Лепестки» $P_i = S_i \setminus C$ попарно не пересекаются: $P_i \cap P_j = \emptyset$ для всех $i \neq j$

Если семейство — подсолнечник, выведите:
- `YES`
- Размер ядра $|C|$
- Размеры всех лепестков $|P_1|, |P_2|, \dots, |P_n|$

Иначе выведите `NO`.

---

## Формат входных данных

Первая строка содержит целое число $n$ ($2 \le n \le 2 \cdot 10^5$) — количество множеств.

Далее следуют $n$ строк: в $i$-й строке сначала дано $k_i$ ($0 \le k_i \le 2 \cdot 10^5$) — размер множества, затем $k_i$ различных целых чисел $x$ ($1 \le x \le 10^9$) — элементы $S_i$.

**Гарантируется**, что суммарное число всех элементов $\sum k_i \le 2 \cdot 10^5$.

---

## Формат выходных данных

Если семейство — подсолнечник, выведите:
```
YES
|c|
|p1| |p2| ... |pn|
```
где:
- $|c|$ — размер ядра (одно число)
- $|p_1|, |p_2|, \dots, |p_n|$ — размеры лепестков ($n$ чисел через пробел)

Иначе выведите:
```
NO
```

---

## Примеры

### Пример 1
**Ввод:**
```
3
3 1 2 3
2 2 3
4 0 2 3 5
```
**Вывод:**
```
YES
2
1 0 2
```
*Пояснение:*
- Ядро $C = \{2, 3\}$, $|C| = 2$
- Лепестки: $P_1 = \{1\}$ (размер 1), $P_2 = \emptyset$ (размер 0), $P_3 = \{0, 5\}$ (размер 2)

### Пример 2
**Ввод:**
```
3
2 1 2
2 2 3
2 1 3
```
**Вывод:**
```
NO
```
*Пояснение:* Не является подсолнечником, так как пересечения пар множеств различны:
- $S_1 \cap S_2 = \{2\}$
- $S_1 \cap S_3 = \{1\}$
- $S_2 \cap S_3 = \{3\}$

---

## Примечания

1. Ядро $C$ может быть пустым множеством ($|C| = 0$)
2. Лепестки также могут быть пустыми
3. Важно, что для **любых** двух различных множеств их пересечение должно быть равно $C$
4. Лепестки (элементы, не входящие в ядро) должны быть **попарно не пересекающимися**
