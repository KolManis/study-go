
# Пересечение множеств

**Ограничение времени:** 1 с  
**Ограничение памяти:** 64.0 Мб  

---

## Условие задачи

Даны несколько множеств целых чисел.  
Необходимо найти количество различных элементов, которые встречаются **во всех множествах одновременно**.

---

## Формат входных данных

Первая строка содержит число \( n \) (\( 1 \le n \le 1000 \)) — количество множеств.

Далее следуют \( n \) строк, описывающих множества.  
В \( i \)-й строке сначала указано число \( k_i \) (\( 0 \le k_i \le 1000 \)) — размер множества, затем \( k_i \) различных целых чисел \( x \) (\( 1 \le x \le 10^6 \)) — элементы множества.

---

## Формат выходных данных

Выведите одно число — размер пересечения всех данных множеств.

---

## Примеры

### Пример 1
**Ввод:**
```
3
3 1 2 3
4 2 3 4 5
2 2 3
```
**Вывод:**
```
2
```
*Пояснение:* пересечение — элементы 2 и 3.

### Пример 2
**Ввод:**
```
3
4 1 2 3 4
3 2 3 5
5 2 3 4 6 7
```
**Вывод:**
```
2
```
*Пояснение:* пересечение — элементы 2 и 3.
