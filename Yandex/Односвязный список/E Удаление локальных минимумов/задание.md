# Удаление локальных минимумов

**Ограничение времени:** 1 секунда  
**Ограничение памяти:** 256.0 Мб  
---

## Условие задачи

Дан массив `a`, состоящий из `n` чисел.

Позиция `i` называется **локальным минимумом**, если для неё выполняются условия:
- `2 ≤ i ≤ n − 1` (элемент не является первым или последним)
- `aᵢ₋₁ > aᵢ < aᵢ₊₁` (значение элемента строго меньше обоих соседей)

Требуется удалить из массива все элементы, стоящие в позициях локальных минимумов, определённых **относительно исходного массива**.

---

## Формат ввода

Первая строка содержит единственное число `n` — количество чисел в массиве.

Вторая строка содержит `n` чисел `aᵢ`, разделённых пробелами, где `aᵢ` — число на `i`-й позиции в массиве `a`.

**Ограничения:**
- `3 ≤ n ≤ 10⁵`
- `1 ≤ aᵢ ≤ 10⁵` для всех `1 ≤ i ≤ n`

---

## Формат вывода

Первая строка должна содержать одно число `k` — количество элементов в массиве, которые остались после удаления.

Вторая строка должна содержать `k` чисел, разделённых пробелами — оставшиеся элементы массива `a` в исходном порядке (за исключением удалённых локальных минимумов).

---

## Примеры

### Пример 1
**Ввод**
```
5
1 3 2 5 4
```
**Вывод**
```
4
1 3 5 4
```
*Пояснение:* Локальный минимум — элемент `2` на позиции 3 (`3 > 2 < 5`). После его удаления остаются элементы `1, 3, 5, 4`.

### Пример 2
**Ввод**
```
5
3 2 1 2 3
```
**Вывод**
```
4
3 2 2 3
```
*Пояснение:* Локальный минимум — элемент `1` на позиции 3 (`2 > 1 < 2`). После его удаления остаются элементы `3, 2, 2, 3`.

### Пример 3
**Ввод**
```
6
5 3 1 2 4 6
```
**Вывод**
```
5
5 3 2 4 6
```
*Пояснение:* Локальный минимум — элемент `1` на позиции 3 (`3 > 1 < 2`). После его удаления остаются элементы `5, 3, 2, 4, 6`.

### Пример 4
**Ввод**
```
7
2 1 2 1 2 1 2
```
**Вывод**
```
4
2 2 2 2
```
*Пояснение:* Локальные минимумы — элементы `1` на позициях 2, 4 и 6. После их удаления остаются элементы `2, 2, 2, 2`.
