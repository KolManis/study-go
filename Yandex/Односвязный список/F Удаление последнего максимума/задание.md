# Минимум слева

## Условие задачи

Дан массив `a`, состоящий из `n` чисел.

Для каждой позиции `i` (где `1 ≤ i ≤ n`) вычислите величину  
```
bᵢ = min(a[1], a[2], …, a[i])
```

Другими словами, `bᵢ` — это минимальный элемент массива `a` на префиксе от первого до `i`-го элемента включительно.

---

## Формат ввода

Первая строка содержит единственное число `n` — количество чисел в массиве.

Вторая строка содержит `n` чисел `aᵢ`, где `aᵢ` — число на `i`-й позиции в массиве `a`. Числа разделены пробелами.

**Ограничения:**
- `1 ≤ n ≤ 10⁵`
- `1 ≤ aᵢ ≤ 10⁵` для всех `1 ≤ i ≤ n`

---

## Формат вывода

Выведите `n` чисел `bᵢ`, разделённых пробелами, в порядке возрастания `i` (от `b₁` до `bₙ`).

---

## Пример

**Ввод**
```
5
5 2 3 4 1
```

**Вывод**
```
5 2 2 2 1
```
